# Whatsapp Bot OpenAI NodeJS

Turn your WhatsApp into an AI chatbot integrated by OpenAI

## NPM Installation
```sh
git clone https://github.com/fitri-hy/wabot-openai.git
cd wabot-openai
npm i wabot-openai
```

## Setting .env
```sh
API_KEY = "you apiKey"
MODEL = "gpt-3.5-turbo-1106"
COMMAND = "Command for AI."
PHONE = "628xxxxxxx"
PREFIX = ".gpt"
```

## Run
`node index.js`
